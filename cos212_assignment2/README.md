## Usage (Task 1)
1. `./configure.sh`
2. Move `black-fitch-makefile` to your source directory:

	```

	├── CircularList.C
	├── CircularList.h
	├── Deque.C
	├── Deque.h
	├── main.C
	├── makefile
	├── package.sh
	├── Queue.C
	├── Queue.h
	├── Stack.C
	├── Stack.h
	```

3. Test with `make -f black-fitch-makefile task1`
4. Update with `make -f black-fitch-makefile pull`
